# Heart Disease Predictor
A machine learning web app to predict heart disease risk using XGBoost.

## Features
- 94.53% AUC-ROC score
- SHAP explainability charts
- Interactive risk gauge

## Run locally
pip install -r requirements.txt
python train.py
